On first release in 1995 the only vinyl release of Wasps' Nest was a box set containing four seven inch EPs with all the tracks (and one bonus one) across them. Mean-ness and my full embrace of CD media meant that I didn't buy that but bought the [CD that I've already posted about](/2025/07/17/my-record-collection-the-6ths-wasps-nest-cd/), In 2017 Captured Tracks released the album on an LP - this is that.

{% ahfowimage "https://media.fullofwishes.co.uk/00-misc/my-record-collection/the-6ths-wasps-nest-lp.jpg" "The 6ths - Wasps' Nest" %}

The LP came in white vinyl and sadly didn't include the extra track it did howver have a larger booklet than the CD which meant that it could include the artist self-portraits that had been included in the box set but were missing from the CD package. Some were lovely and detailed, some were realistic, some were cartoon-y, some minimalist, some impressionist... and then there was Dean's:

{% ahfowimage "https://media.fullofwishes.co.uk/00-misc/my-record-collection/the-6ths-wasps-nest-dean-wareham-portrait.jpg" "Dean Wareham self-portrait" %}

I think I said before how much I love this album and it really is a very beautiful thing, and doesn't have a dud on it!

 - *Catalogue Number:* AHFOW 12/073
 - *Artist:* The 6ths
 - *Title:* Wasps' Nest
 - *Notes:* White vinyl. Includes an 8" square booklet
 - *Format:* LP
 - Bought direct from Captured Tracks in 2017 for $25.00 (plus postage which was tolerable back then!)
